# OHMEALS React Dashboard

This folder will contain the React admin dashboard application.

## Setup (À faire)

```bash
npx create-react-app .
# or
npx create-vite@latest . --template react
```

## API Endpoints disponibles

| Méthode | Endpoint | Description |
|---------|----------|-------------|
| GET | `/api/admins` | Liste des admins |
| POST | `/api/admins` | Créer un admin |
| PUT | `/api/admins/:id` | Modifier un admin |
| DELETE | `/api/admins/:id` | Supprimer un admin |
| GET | `/api/products` | Liste des produits |
| POST | `/api/products` | Créer un produit |
| PUT | `/api/products/:id` | Modifier un produit |
| DELETE | `/api/products/:id` | Supprimer un produit |
| GET | `/api/orders` | Liste des commandes |
| POST | `/api/orders` | Créer une commande |
| PUT | `/api/orders/:id` | Modifier une commande |
| DELETE | `/api/orders/:id` | Supprimer une commande |
| GET | `/api/stats` | Statistiques dashboard |
